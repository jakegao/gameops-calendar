主界面交互说明
以暗区突围「存在感美学」为基底，整体采用深黑底色 + CRT军事终端视觉语言。
模拟90年代旧式军事电子屏幕的操作体验，功能信息优先，避免过度装饰。
A

战术终端头部 HEADER

左侧黄铜色竖条标识活动品牌。英文终端代号 + 中文名称双层标题。
右上角显示「暗区行动令」余额，采用等宽字体模拟LED数字显示。
下方活动倒计时条，使用 "OPERATION WINDOW" 军事行动窗口概念。

B

五行扫描区 SCAN ZONE

核心交互区域。暗黑背景 + 网格线 + 雷达扫描动效，持续旋转的扫描线营造监控感。
五行元素以菱形军事徽章排列，各自颜色独立发光。
星级用★标记当前等级，点击单个元素可查看详情浮层。
已激活元素有脉冲光晕动效，未激活元素低饱和度待机。

C

扫描启动按钮 ACTION BUTTON

主CTA按钮，黄铜色边框 + 内部光线扫过动效。
按下触发：CRT信号干扰(0.3s) → 雷达扫描(0.7s) → 元素锁定(0.5s) → 结果揭示(0.5s)。
10连扫描按钮在下方，精简动效每次0.5秒，最后汇总展示。

D

底部功能栏 FUNCTION BAR

三个功能入口并排：「强化追加」进入升星博弈界面，「物资兑换」进入保底商店，「奖励预览」查看全部奖品。
采用暗区突围标准的深灰卡片样式，悬停时边框亮起黄铜色。

E

物资凭证进度 SUPPLY PROGRESS

底部常驻进度条，显示累计获得的暗区物资凭证数量。
满972枚可兑换任意一套五行战术套装。黄铜色进度条 + 尾部发光点。
每次扫描/碎裂均产出凭证，保证玩家持续积累保底。
